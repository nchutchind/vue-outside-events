v-click-outside, passing an object (vue@3.2.40)

Click around... see the colors change and labels pop. See the data value get passed with the directive.

Vue Directives can have data bound to them once, at run-time when they are created. If you bind a variable to the directive and then change the variable, the directive will maintain the old value when it calls the handler.

If you use a composable or ref, you can get updates. In this example, a and b are statically bound. c, d, and e are bound with a ref. f, g, h, and i use composables and template refs.

v-click-outside example

		

		<script>
			export default {
				methods: {
					onClickOutside(e, el, extras) {
						console.log("onClickOutside");
						console.log("Event:", e);
						console.log("Element clicked on:", e.target);
						console.log("Element clicked outside of:", el);
						console.log("Extras:", extras)
					}
				}
			}
			</script>
